Administrador de suscripción de gimnasio:

Esta aplicación permite llevar la gestión de las suscripciones de un gimnasio, llevando el control de las fechas de pago y sus días restantes, añadiendo colores para la identificación de posibles atrasos.

Gestor de apartados:

Aplicación que getiona los pedidos de un negocio de maquillaje, agregando a cada apartado un nombre, fecha de pedido y fecha de vencimientos, ademas de la gestión de los pagos. Esta aplicación utiliza Firebase como base de datos lo que permite ver y gestionar los apartado desde cualquier celular que tenga instalada la app.

Mi Tianguis:

Esta aplicación pretende juntar la publicidad de negocios locales para categorizarlos y tener una facil comunicación con ellos. Esto nos da la ventaja de comparar, ver la ubicación exacta del negocio y descubrir nuevos productos. Este proyecto se encuentra en desarrollo y pretende usar Firebase como base de datos.

Cifrado Cesar:

Con esta aplicación puedes escribir un mensaje, elegir una llave y cifrar dicho mensaje, puedes copiar el mensaje cifrado y mandarlo por cualquier medio, el usuario receptor pega dicho mensaje en la aplicación y poniendo la llave correcta puede descifrar el mensaje.
